# Чек-лист: реквизиты и приложения

Справочный материал к скиллу `otmena-sudebnogo-prikaza`. Читать, когда готовишь
документы и нужно проверить, что ничего не упущено.

## Реквизиты для возражений

Обязательно выяснить у клиента и указать в документе:

- [ ] мировой судья: номер судебного участка и район/город;
- [ ] номер дела (номер судебного приказа);
- [ ] дата вынесения судебного приказа (если известна);
- [ ] взыскатель: наименование/ФИО, адрес (из приказа);
- [ ] должник: ФИО, адрес, дата рождения (для идентификации в ИП);
- [ ] сумма и период взыскания по приказу;
- [ ] дата получения копии приказа должником (ключ для срока).

## Приложения к возражениям

- [ ] копия судебного приказа (если есть на руках);
- [ ] доказательство даты получения/узнавания (конверт с отметкой почты, скрин
      Госуслуг, постановление пристава) — особенно при пропуске срока;
- [ ] копия постановления о возбуждении исполнительного производства (если есть);
- [ ] документы в обоснование спорности долга (для будущего иска, не для отмены).

## Когда нужно ходатайство о восстановлении срока

Подавать вместе с возражениями, если 10 дней с момента получения копии приказа
истекли. Основание — уважительность причины пропуска. Приложить доказательство,
**когда фактически** получил/узнал:

- скрин/выписка из Госуслуг с датой;
- конверт/трек «Почты России» с датой вручения или возврата;
- дата постановления пристава / дата списания по выписке банка.

## После отмены приказа

- [ ] получить определение об отмене судебного приказа;
- [ ] заявление приставу о прекращении исполнительного производства (приложить
      определение);
- [ ] заявление о повороте исполнения — вернуть уже списанное;
- [ ] предупредить клиента: взыскатель может подать иск в общем порядке.

## Частые возражения по существу (для будущего иска по ЖКУ)

Не нужны для самой отмены, но фиксируй сразу — пригодятся в исковом споре:

- пропуск срока исковой давности по периодическим платежам;
- неверный расчёт (тариф, площадь, число проживающих, период);
- ненадлежащий должник (собственник vs наниматель vs зарегистрированный);
- двойное начисление, неучтённые платежи;
- спор по пеням (период моратория, ключевая ставка).
