# Contributing

Спасибо за интерес к LawMatic Skills!

## Как добавить скилл

1. Создайте ветку от `main`.
2. Добавьте папку в `skills/` с файлом `SKILL.md`.
3. Убедитесь, что frontmatter содержит поля `name` и `description`.
4. Обновите таблицу в `README.md`.
5. Откройте pull request с кратким описанием: для какой задачи скилл, когда агент должен его применять.

## Требования к скиллам

- `name` — уникальный, lowercase, дефисы (kebab-case).
- `description` — полное описание того, **что** делает скилл и **когда** его использовать.
- Без секретов, API-ключей и внутренних URL.
- Скилл должен быть самодостаточным — агент не должен знать контекст других проектов LawLabs.

## Лицензия

Внося изменения, вы соглашаетесь, что они распространяются под [MIT License](LICENSE).
