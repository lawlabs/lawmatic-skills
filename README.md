# LawMatic Agent Skills

Agent Skills для юридической работы в [Cursor](https://cursor.com), Claude Code и других AI-агентах.

Скиллы — это инструкции, которые учат агента выполнять специализированные задачи: анализ документов, расчёт сроков, работа с судебной практикой и другие юридические workflow.

## Установка

```bash
npx skills add lawlabs/lawmatic-skills
```

Из интерактивного меню можно выбрать конкретные скиллы для установки.

### Cursor

Скиллы можно установить вручную, скопировав нужную папку из `skills/` в `.cursor/skills/` вашего проекта или в `~/.cursor/skills/` для глобального доступа.

## Структура репозитория

```
skills/
└── skill-name/
    ├── SKILL.md       # обязательный — инструкции и метаданные
    ├── reference.md   # опционально — справочные материалы
    └── scripts/       # опционально — вспомогательные скрипты
```

## Создание скилла

1. Создайте папку в `skills/` с именем в kebab-case (например, `court-deadline-check`).
2. Добавьте `SKILL.md` с YAML frontmatter:

```markdown
---
name: court-deadline-check
description: Проверка процессуальных сроков. Использовать когда нужно рассчитать или проверить срок подачи документов в суд.
---

# Court Deadline Check

[Инструкции для агента]
```

3. Откройте pull request или issue с описанием скилла.
